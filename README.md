# Cecília Tattoo Landing Page

Landing page premium dark para Cecília Anacleto, tatuadora especialista em Old School.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Google Fonts: Kanit

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Observações

- A hero usa personagem 3D com animação suave.
- Os objetos 3D da hero flutuam com tempos diferentes.
- CTAs apontam para WhatsApp e Instagram.
- As imagens estão em `src/assets/images`.
- Ajuste `brand.location` em `src/data/siteContent.ts` quando tiver a localização real.
